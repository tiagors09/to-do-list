# Counter

Task manager that allows the control of states by color.

White - to do
Green - doing
yellow - done
Red - fail

## Installation

Clone repository.

```bash
git clone https://github.com/tiagors09/to-do-list.git
```

Open project dir.

```bash
cd to-do-list
```

Use the package manager [npm](https://www.npmjs.com/) to install counter dependencies.

```bash
npm install
npm run dev
```
